# JavaScript 基础知识

JavaScript 是现代 Web 开发的核心技术之一。

## 变量声明

```javascript
const message = "Hello World";
let count = 0;
var oldStyle = "不推荐使用";
```

## 函数定义

```javascript
// 箭头函数
const add = (a, b) => a + b;

// 传统函数
function multiply(a, b) {
  return a * b;
}
```
