# 感配色偶得灵感 借AI初启博客

## AI 缘起


这个博客的风格颜色基本调是我在书店看到王立群新印版的《王立群读史记》系列的配色。本身这版的基调就早已经做好，后面的详情搜索等页面，我懒得去做，就搁置下了。现在基于 AI，我几乎没有写一行代码，就把后面的内容都基于 AI 来完成。用的是 Claude 4，是真鸡巴好用，在加上 MCP，目前是我用的最爽的一个 AI。所以这算是我写博客的一个缘起吧。



## Claude 国内白嫖

使用AI 最头痛的就是怎么调用了。Claude 又是国外的一个巨头，国内访问不了，于是我找了很多方法，还找同事想一起开通一个账号，后面同事的账号被风控了。后面在补充怎么白嫖吧。


## MCP 邮箱服务

在使用 AI的过程中，我玩了一下 coze，有个叫做 coze 空间的东西，可以上传 MCP 服务，基于本地的 stdio 的形式。于是我又一个鬼点子，就是搞一个邮箱MCP服务，然后通过爬虫MCP把数据通过邮箱发送出来，微信有个QQ 邮箱提醒，当绑定了 QQ 邮箱，就可以在微信收到邮件提醒了。于是我通过Claude 写一个邮箱MCP服务，上传到 coze，发邮件的时候总是报超时，初步判断是服务器把邮箱的端口都禁用了，我尝试在 coze 的反馈群里反馈，但是没有人应我，我在想别的方法。
